    if-else

    In python, there are two kinds of if-else expressions:

    `

    inline if-else

    'yes' if True else 'no'

    if-else block

    if True: 'yes' else: 'no' `

    Note that the body of an inline if-else always returns a value, while an if-else block will contain a number of expressions to run. This is an important distinction, since it means you can use has_equal_value on the body of a inline if-else, but not an if-else block.
